Nikola Tesla Chatbot
A chatbot interface built with React/express that enables the user to converse with Nikola Tesla via a JSON scripting engine and
a mini huggingface model API (AI functionality to be fixed).
This is an experimental project developed for university coursework and to learn frontend/backend programming basics.

Learnt:
- Components and react hooks
- Async programming
- Basic UI/UX design
- Server routing
- External API linkage
- How to use JSON
